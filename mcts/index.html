What it catches

Three classes of supply-chain attack, caught before runtime.

01
Tool poisoning

Tool poisoning: hidden instructions, deceptive descriptions, schema-divergent behavior.

02
Prompt injection

Prompt injection: injected directives in tool outputs or manifests that hijack the agent.

03
Credential leak

Credential leak: tools that read secrets, tokens, or keys and route them to unexpected destinations.
