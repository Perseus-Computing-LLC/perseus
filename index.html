✦ Our solution

Resolve your environment before the session opens.

Perseus is a pre-processor. You write directives once in a markdown source file; at render time Perseus resolves them — git state, service health, test status — into a plain file your assistant reads. One pip install, one command. The assistant opens knowing exactly what's running.

~/project
$ pip install perseus-ctx
$ perseus render
 
✓ 4 directives resolved in 31ms
✓ wrote CLAUDE.md (3.2 KB)

# exactly what's running.
