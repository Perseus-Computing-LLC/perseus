Persistent memory engine · v2.6.0

Your agent remembers, across every session.

Mimir is a local, encrypted memory server for AI agents. Facts, decisions, and context persist in SQLite on your machine. They are searchable by keyword and meaning, they decay by confidence, and they never leave your control.

How memory works

One local store, layered by recency and confidence.

Every memory is written to an encrypted SQLite file with an embedding and a confidence score. On recall, Mimir blends keyword and vector hits, weights by freshness, and returns a compact ranked set.
