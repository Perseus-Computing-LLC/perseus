§ 03

Quickstart Setup

Perseus Cloud is a hosted platform, but every component is MIT-licensed and runs locally too. Install the core tools and point them at your cloud workspace in under a minute.

1. Install Perseus
# Install the core live context engine
pip install perseus-ctx

# Verify the installation
perseus doctor
2. Install Mimir
# Install the persistent memory sidecar
pip install mimir

# Or one-shot bootstrap (Rust binary)
curl -sSL https://raw.githubusercontent.com/Perseus-Computing-LLC/mimir/main/scripts/bootstrap.sh | bash
3. Connect to Perseus Cloud
# Point your local tools at the cloud workspace
perseus quickstart --cloud

# Or configure manually in .perseus/config.yaml:
perseus_cloud:
  workspace: "your-workspace-id"
  endpoint: "https://cloud.perseus.observer"
  mimir:
    remote: true
  plutus:
    providers: ["anthropic", "openai"]
