# Patent Artifacts

Private working materials for U.S. provisional application 64/069,842,
`RESOLVE-BEFORE-CONTEXT PIPELINE FOR AI LANGUAGE MODEL CONTEXT ASSEMBLY`.

These files are not legal advice and are not automatically part of the USPTO
record. They are working notes to preserve the technical narrative, prepare
future filings, and keep the invention story concrete.

## Files

- `technical-disclosure.md` - engineering disclosure of the system, mechanisms,
  inventive concepts, and implementation support.
- `figures.md` - draft figure list with Mermaid diagrams and figure narratives.
- `claim-concepts.md` - provisional claim-style concepts for later attorney or
  agent drafting.
- `support-checklist.md` - private checklist for tracking what has support in
  the disclosure and what still needs detail.
- `cover-sheet-data.md` - plain-language data to help complete PTO/SB/16 or an
  equivalent provisional cover sheet.

## Near-Term Use

Keep these private. If the USPTO later asks only for a missing cover sheet, use
`cover-sheet-data.md` to complete the official form. If you decide to pursue a
non-provisional before May 19, 2027, use the other files as the starting point
for a formal specification and claim set.
