§ 01

What Plutus Does

Plutus monitors your API credit balances across every LLM provider you use. It shows remaining credits, calculates your burn rate, and predicts when you'll run out — all before you open your next context window.

📊

Balance Tracking

Poll every provider in parallel. See remaining credits, total spent, and per-model breakdowns. Anthropic, OpenAI, Google, AWS, Azure — one command, all answers.

📈

Burn Rate Analysis

Plutus tracks your daily spend velocity and computes a rolling burn rate. Know not just what you have, but how fast you're spending it.

🔮

Depletion Forecasting

ML-driven forecasts predict your credit exhaustion date. Get warnings 7, 3, and 1 day before you hit zero — with configurable thresholds.

🔔

Usage Alerts

Configurable alert thresholds. Get notified via Slack, Discord, or email when a provider balance drops below your defined limit.

🔌

Pluggable Providers

Provider adapters are simple Python plugins. Add a new provider in ~50 lines. Anthropic, OpenAI, Google, AWS, and Azure are built-in.

🪞

Perseus Native

Plutus integrates directly with Perseus context directives. Use @plutus to inject your current credit status into every session.

§ 03

Quickstart Setup

Get Plutus running in under 60 seconds. Install, configure your provider keys, and run your first balance check.

1. Install Plutus
# Install Plutus via pip
pip install plutus-credit-monitor

# Verify the installation
plutus --version
2. Configure Providers
# Quick setup wizard — prompts for API keys
plutus init

# Or configure manually in ~/.plutus/config.yaml:
providers:
  anthropic:
    api_key: "$ANTHROPIC_API_KEY"
  openai:
    api_key: "$OPENAI_API_KEY"
  google:
    project_id: "your-gcp-project"
  aws:
    region: "us-east-1"
  azure:
    endpoint: "https://your-resource.openai.azure.com"
3. Run Your First Check
# Check balances across all configured providers
plutus status

# Output:
# ┌────────────┬──────────────┬────────────┬──────────────┐
# │ Provider   │ Remaining    │ Burn Rate  │ Days Left    │
# ├────────────┼──────────────┼────────────┼──────────────┤
# │ Anthropic  │ $432.18      │ $18.42/day │ 23 days      │
# │ OpenAI     │ $215.00      │ $12.65/day │ 17 days      │
# │ Google     │ $300.00      │ $5.20/day  │ 57 days      │
# │ AWS        │ $500.00      │ $8.90/day  │ 56 days      │
# │ Azure      │ $150.00      │ $22.00/day │ 6 days ⚠     │
# └────────────┴──────────────┴────────────┴──────────────┘

# Watch continuously (updates every 60s)
plutus watch --interval 60
4. Integrate with Perseus
# Add a @plutus directive to your .perseus/config.yaml
directives:
  - "@plutus status --summary"

# Now every Perseus context window opens with your credit status
perseus render
