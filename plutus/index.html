What it does

Every provider's spend in one ledger.

Live balances

Pulls remaining credit from each provider on an hourly cadence, with no manual dashboard checks.

Unified spend ledger

Normalizes usage across providers into a single local record you can query, export, and audit.

Burn-rate alerts

Set thresholds per provider and get warned locally before a key runs dry mid-session.

One command

A balance check that fits in your prompt.

Run it standalone, or let Perseus pull live balances into your context file so your assistant knows the budget it's working against.
