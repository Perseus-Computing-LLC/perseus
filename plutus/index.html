✦ Plutus · LLM spend monitoring

Know your burn before the credits run out.

Plutus is a CLI dashboard that tracks API credit balances across every LLM provider in real time — burn-rate analysis, depletion forecasting, and usage alerts — so a stalled agent run is never a billing surprise.

✦ What it does

Every provider's balance, in one view.

◉

Live balance tracking

Real-time credit balances across all configured providers, refreshed hourly — one dashboard, no portal-hopping.

↗

Burn-rate analysis

Spend velocity per provider and per project, so you see which workloads are eating the budget.

◔

Depletion forecast

7-day predictive modeling tells you when you'll run dry — before an agent run stalls mid-task.

⚐

Usage alerts

Configurable thresholds fire before you hit a wall. No more silent 429s in production.

⊕

Custom providers

Add any provider in ~50 lines of Python. The plugin surface is small by design.

Set up in under a minute.

Point Plutus at your provider keys once. From then on, plutus status is your single source of truth for AI spend — locally, or surfaced into agent context through Perseus.
