Who We Are

Perseus Computing is a U.S. small business building the infrastructure layer that makes AI agents reliable, auditable, and deployable inside disconnected and classified environments. Our software runs fully on-premise with zero cloud dependencies and ships under an MIT license with a published SBOM — procurement-ready, no vendor lock-in.

Core Competencies

Live Context Engine (Perseus). Renders live system state — repository, tickets, deployment status, prior decisions — into verifiable, cryptographically signed Markdown facts and injects them into an AI agent's context before it reads. Eliminates the multi-turn context drift, hallucination, and repeated work that degrade AI-assisted software development.

Auditable Memory Layer (Perseus Vault). Bitemporal, on-premise memory and retrieval engine for AI agents — every stored fact is signed, versioned, and traceable, giving regulated and classified programs a defensible record of what an agent knew and when.

Verifiable Efficiency Metering (Plutus). Measures and certifies the token- and dollar-cost reduction delivered by the context layer, producing tamper-evident before/after artifacts rather than vendor estimates.

C3BM Applicability

As C3BM software factories (Kessel Run; ABMS Software & Applications) adopt AI-assisted development, they face three gaps Perseus closes at the infrastructure layer: tools losing context between sessions, no support for disconnected/classified environments, and unauditable agent output. Perseus makes AI-assisted delivery faster and auditable inside air-gapped networks — a fit from rapid experimentation (e.g., Operational Response Team) through production software delivery.
